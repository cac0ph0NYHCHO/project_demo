# STM32F103 无阻塞多任务智能控制系统
基于 STM32F103C8T6 的模块化、非阻塞、状态机架构综合项目

## 项目简介
本项目基于 STM32 标准库开发，实现**无阻塞多任务嵌入式系统**，  
包含按键输入、红外中断、CAN 总线通信、OLED 显示、PWM 呼吸灯、LED 状态机、串口日志输出等功能。  
全程不使用 delay 阻塞，采用**定时器时间戳 + 状态机架构**，  
系统响应快、稳定性高、代码结构清晰，符合企业级嵌入式开发规范。

---

## 硬件平台
- 主控：STM32F103C8T6
- 外设：
  - 按键模块
  - 红外避障传感器
  - OLED 显示屏（I2C/SPI）
  - CAN 收发器
  - LED + PWM 呼吸灯

---

## 实现功能
✅ 非阻塞系统架构（无 delay）  
✅ 蓝色 PWM 呼吸灯（平滑亮度调节）  
✅ 红色 LED 状态机（熄灭 / 慢闪 / 快闪）  
✅ 按键消抖 + 触发 CAN 数据发送  
✅ 红外传感器 EXTI 外部中断触发紧急告警  
✅ CAN 总线收发 + OLED 实时显示  
✅ USART 串口日志输出（printf 重定向）  
✅ 日志可保存到电脑 TXT 文件  
✅ 超时自动恢复状态

## 使用说明
1. Keil MDK 编译下载
2. 开启 Use MicroLib
3. 串口助手波特率 9600
4. 按键 1 发送 CAN 数据
5. 遮挡红外传感器触发紧急告警模式
